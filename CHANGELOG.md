# CHANGELOG

## [0.3.0] — 2026-04-01

### 新增功能

#### 🔒 PDF 加密下載

- 「💾 輸出」選單新增第三個選項「🔒 加密後下載…」
- 點擊後彈出加密 Modal，輸入密碼與確認密碼（含錯誤提示）
- 使用 pdf-lib `doc.save({ userPassword, permissions })` 實作加密
- 加密設定：允許列印（高解析度）、允許填表，禁止修改、禁止複製

#### 🧹 標示橡皮擦

- 標示工具列新增「🧹 橡皮擦」工具按鈕
- 支援**點擊**與**拖曳**兩種操作方式，連續刪除
- 各類型形狀的碰撞判斷邏輯：
  - **畫線（pen）/ 螢光筆（marker）**：點到折線段距離 ＜ 筆寬 / 2 + 15px
  - **矩形（rect）**：點在 AABB 範圍 + 15px margin 內
  - **橢圓（ellipse）**：橢圓方程式 + 15px margin
- 文字標注維持原本「點擊標記刪除」方式，不受橡皮擦影響
- 切換至橡皮擦時游標自動變為 `cell` 樣式

## [0.2.0] — 2026-04-01

### 新增功能

#### 🎨 標示標色工具

- 標注面板新增五種繪圖工具：**文字、簽字筆、螢光筆、矩形、橢圓**
- 顏色快選：黑、白、紅、藍、綠、黃（六色色塊）
- 透明度控制：外框 0–100%、底色 0–100%（底色僅矩形 / 橢圓顯示）
- 輸出至 PDF 時矩形 / 橢圓使用 pdf-lib 原生繪製；簽字筆 / 螢光筆轉 PNG 圖層嵌入

#### ✂️ PDF 拆分（多份輸出）

- 工具列新增「拆分 PDF」按鈕，開啟獨立拆分面板
- 可設定多段頁碼範圍（第幾頁 ～ 第幾頁），點「＋ 新增區段」追加
- 剩餘未涵蓋頁面自動合併為最後一份，即時顯示計算結果
- 支援 File System Access API 選資料夾批次儲存；不支援時逐一下載

#### 💾 合併儲存詢問

- 合併後透過 `showSaveFilePicker` 詢問儲存路徑與自訂檔名
- 不支援 File System Access API 時改以 `prompt()` 輸入檔名後下載

#### 🖼️ 圖片轉 PDF

- 載入與拖放新增支援 `.jpg`、`.jpeg`、`.png`
- 使用 pdf-lib `embedJpg` / `embedPng` 嵌入；超過 A4 自動等比縮放，否則維持原尺寸

## [0.1.0] — 2026-03-31

### 初始版本

#### 核心功能

- **頁面管理**：PDF 縮圖預覽、拖曳重新排序（Sortable.js）、旋轉（±90°）、勾選頁面
- **合併**：多個 PDF 合併成一個，可拖曳調整順序，支援資料夾批次載入
- **分割**：勾選頁面後單獨輸出為一份 PDF
- **文字標注**：點擊頁面放置文字，可設定大小、顏色；點擊標記刪除
- **表單設計**：拖曳畫出欄位框、設定欄位代碼與說明標籤，輸出含可填寫欄位的 PDF
- **Word 轉 PDF**：上傳 `.docx`，透過 mammoth.js → html2canvas → jsPDF 轉換後載入工具

#### 技術架構

- 純前端單一 HTML 檔，無需後端或安裝
- 支援 File System Access API（資料夾讀寫）；不支援時自動降級為下載模式
- CDN：pdf.js 3.11.174 · pdf-lib 1.17.1 · Sortable.js 1.15.2 · mammoth.js 1.8.0 · html2canvas 1.4.1 · jsPDF 2.5.1
- 提供 `build-offline.js` 打包腳本，產生離線版（內嵌所有 CDN library）
