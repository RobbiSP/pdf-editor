## [0.7.0-dev] — 2026-04-02（舊開發記錄）

### 修復

#### 🖼️ PDF 左邊無法顯示
- 修復縮放時 PDF 左側內容無法滾動查看的問題
- **根因**：CSS 使用 `display:flex; justify-content:center` 導致 overflow:auto 無法正確工作
- **解決方案**：改為 `margin:0 auto` 方式居中，保留完整滾動範圍

#### 🖱️ Space+滑鼠平移自動跳回
- 修復使用 Space 鍵+滑鼠拖曳時，拉到最下方會自動跳回最上面的問題
- **根因**：`e.preventDefault()` 位置不對，Space 鍵重複事件仍觸發默認滾動
- **解決方案**：將 `preventDefault()` 移到 `if (!spaceDown)` 外，但保留標誌檢查

#### ✏️ 已新增文字無法選取再編輯
- 修復文字框在切換工具模式後變成無法編輯/移動的問題
- **根因**：`contentEditable` 被工具模式條件控制，導致在「選取模式」變成唯讀
- **解決方案**：文字框改為總是 `contentEditable=true`，編輯/移動行為改由事件處理器控制

#### 🔤 導出 PDF 文字模糊
- 提升文字渲染品質，改用 2 倍 DPI (High DPI) 解析度
- 文字標註與文字框現在在高解析度 canvas 上繪製，再嵌入 PDF
- 視覺效果更清晰，特別是小字體

### 新增功能

#### ✨ 文字框 UX 改進（控制框 + 浮動工具欄 + 拖動調整大小）

- **藍色實線選取框**：選中文字框時顯示 `outline:2px solid #4c6ef5 + box-shadow`，清楚表示可編輯狀態
- **浮動工具欄**：選中文字框時在頂部顯示
  - 當前字體大小（可點擊輸入精確值 6~120px）
  - 6色色塊快選（黑、白、紅、藍、綠、黃）
  - 刪除按鈕（✕）
- **右下角拖動點**：藍色 12×12px 方形
  - 上下拖動即時調整字體大小
  - 字體大小數字即時更新
  - 支援 6~120px 範圍
- **精確輸入**：點擊浮動工具欄中的字體大小
  - 變為輸入框 → 輸入值 → Enter 確認 / Escape 取消
- **同步更新**：拖移文字框時，浮動工具欄和拖動點位置即時跟隨
- **自動隱藏**：點擊空白處時浮動工具欄與拖動點自動隱藏
- **技術實現**：
  - 使用 event delegation 處理浮動工具欄事件
  - 每次位置更新時重新初始化必要的事件監聽器
  - Inline 編輯字體大小時動態建立 input 並在提交後還原 span

## [0.1.0] — 2026-03-31

### 初始版本

#### 核心功能

- **頁面管理**：PDF 縮圖預覽、拖曳重新排序（Sortable.js）、旋轉（±90°）、勾選頁面
- **合併**：多個 PDF 合併成一個，可拖曳調整順序，支援資料夾批次載入
- **分割**：勾選頁面後單獨輸出為一份 PDF
- **文字標注**：點擊頁面放置文字，可設定大小、顏色；點擊標記刪除
- **表單設計**：拖曳畫出欄位框、設定欄位代碼與說明標籤，輸出含可填寫欄位的 PDF
- **Word 轉 PDF**：上傳 `.docx`，透過 mammoth.js → html2canvas → jsPDF 轉換後載入工具

#### 技術架構

- 純前端單一 HTML 檔，無需後端或安裝
- 支援 File System Access API（資料夾讀寫）；不支援時自動降級為下載模式
- CDN：pdf.js 3.11.174 · pdf-lib 1.17.1 · Sortable.js 1.15.2 · mammoth.js 1.8.0 · html2canvas 1.4.1 · jsPDF 2.5.1
- 提供 `build-offline.js` 打包腳本，產生離線版（內嵌所有 CDN library）
